# Chillow (static site)

This is a simple, static website for **Chillow** (a pillow).

## GitHub Pages (hosting)

### Option A: Project site (recommended for most repos)
1. Create a GitHub repo (example: `chillow`)
2. Upload these files to the repo root (same folder as this `README.md`)
3. In GitHub: **Settings → Pages**
4. Under **Build and deployment**:
   - **Source:** Deploy from a branch
   - **Branch:** `main` (or `master`) and folder `/ (root)`
5. Wait for GitHub to publish, then open the URL it shows (usually `https://<username>.github.io/<repo>/`)

### Option B: User/Org site
1. Create a repo named exactly `YOURUSERNAME.github.io`
2. Put these files in the repo root
3. GitHub Pages will publish at `https://YOURUSERNAME.github.io/`

## Customize
- Edit copy + sections: `index.html`
- Edit colors + layout: `styles.css`
- Edit the checkout placeholder: `checkout.html`
